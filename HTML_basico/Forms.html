Formularios

Los formularios son para pedir datos al usuario (edad, nombre, correo, numero, etc). HTML da unas opciones por defecto de tipo de dato a pedir. Su etiqueta es "form" y se utiliza junto a la etiqueta "imput" para que el usuario pueda ingresar los datos. A la etiqueta imput se le agrega el atributo "type" para especificar el tipo de dato que se esta pidiendo

Tipos de formularios

  • Type=text

    • Crea una casilla para ingresar un texto

  • type=email

    • Crea una casilla ingresar texto pero bajo el formato especifico de correo (@)

  • Type=password

    • Crea una casilla para ingresar texto, pero este se tapa para que no se vea

  • Type=chekbox

    • Pide tildar una casilla

  • Type=radio

    • Pide tildar una de varias opciones

  • Type=file

    • Pide adjuntar un archivo

  • Type=date

    • Pide ingresar una fecha

  • Type=color

    • Pide seleccionar un color

  • Type=submit

    • Boton de "enviar"
