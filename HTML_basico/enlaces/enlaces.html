Etiqueta a

Etiqueta para manteter la ubicacion en un mismo sitio (a de ancla). Esta etiqueta te redirije a un enlace

A la qtiqueta a se le da el atributo funcional href="". Este atributo sirve para crear una referencia de hipervinculo (crear un enlace)

Un atributo tambien puede definir el tamaño, color, etc de una etiqueda. Pero cuestiones de estilo se hacen con CCS

Para hacer que el enlace se habra en otra pestaña, a la etiqueta a se le agrega otro atributo llamado target= y se le define como "_blank"
