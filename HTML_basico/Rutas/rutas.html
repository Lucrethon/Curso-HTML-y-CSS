Manejo de Rutas

Tipos de Rutas

  • Rutas Absolutas

    • Son rutas donde se coloca un recurso y es absoluto que esta ahi, como un enlace de una red social por ejemplo, donde se puede acceder a ella desde cualquier parte de la misma forma

  • Rutas Relativas

    • Son rutas en las que se puede acceder a los recursos que estan en ella de formas diferentes dependiendo de donde te encuentres. Por ejemplo, un archivo .html que se encuentra dentro de una carpeta, solo puede acceder a los recursos de esa carpeta. Para acceder a recursos que se encuentran en otras carpetas, se tiene que hacer un llamado diferente dependiendo de donde se encuentre

Formas de acceso de rutas relativas

  • Si hay que retroceder una carpeta atras

    • href="../nombre_de_archivo"

    Se coloca ../ antes de escribir el nombre del archivo. Si se quiere atrasar mas de una posicion, seria "../../" y asi sucesivamente

    De esa forma retrocedes una carpeta atras. Ejemplo: si se quiere acceder a un archivo que esta en "carpeta_1" desde "carpeta_2", y "carpeta_2" esta contenida en "carpeta_1", entonces se tiene que retroceder una carpeta atras

  • Si hay que avanzar una carpeta

    • href="nombre_carpeta/nombre_de_archivo" (como si fuera una ruta de windows)

    Se coloca el nombre de la carpeta a la que se quiere acceder antes de escribir el nombre del archivo. Si se quiere adelantar mas de una posicion, seria "carpeta1/carpeta2/nombre_archivo"

    Ejemplo: si se quiere acceder a un archivo que esta en "carpeta_2" desde "carpeta_1", y "carpeta_2" esta contenida en "carpeta_1", entonces se tiene que "avanzar" una carpeta
