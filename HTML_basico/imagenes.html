Para cargar una imagen en HTML, se utiliza la etiqueta img y se le da el atributo "source (src="ruta_imagen")" Si la imagen no carga por algun motivo, podemos hacer que se coloque un texto en su lugar con la etiqueta "alt=texto_de_remplazo" Esta etiqueta tambien sirve para dar una descripcion de la imagen quer funciona para el posicionamiento y el SEO
